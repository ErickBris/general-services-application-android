Getting Started

To start with the project you have to perform some simple tasks.
1. Import the project into your Android Studio.
2. Modify admob ads into the project.
3. Changing the basics of project like app name, package name.
4. Reskinning the project with changing the graphic content present in the project.
5. Deploy in the play store.
Yep thats it . These are the only step you need to worry about . Each step is explained in detail given below.

Importing code into Android Studio

To import the code into your android studio do the following steps
  1. Copy the project into android studio project workspace
  2. Select Import Project
  3. Select Project General Services Application
In the dialog, choose the option "Existing Project" and put the path to the  project root path.

Eg - you have put code in C drive then put the path as Local Disk C
In the following list you will be shown the list of projects that you can select Then select General Services Application
Select Ok to import it into the android studio.

Changing Package Name

To change package name 
Right click on your project(General Services Application) in android studio, scroll down to Android Tools, and then click on Refactor-->Rename and change Application Package.

Enter your name and hit Ok

Changing App name

In android studio Go to your project (General Services Application) 
 res->values->strings
then you will see a line like this

  1. <string name="app_name">General Services Application</string>

change this to  

  1. <string name="app_name">YOUR_NAME</string>

Changing Graphic Content

You can change all the graphic content used in this app from the following folder

Go to General Services Application------>>>res

In the res folder all the graphic are present.

The folder setup is as follow

  • Drawable               -Includes graphices used in the App (which is common to all screen). 
  • Drawable-ldpi       - For ~120dpi density Devices
  • Drawable-hdpi       - For ~240dpi density Devices
  • Drawable-mdpi      - For ~160dpi density Devices
  • Drawable-xhdpi      - For ~320dpi density Devices
  • Drawable-xxhdpi    - For ~480dpi density Devices

You should change the content present in Drawable, Drawable-ldpi,Drwable-mdpi,Drawable-xhdpi,Drawable-xxhdpi and Drawable-hdpi for the sake of uniqueness of your app.

 

 

 

Running App on Android Device/Emaluator

Running app on android side is very simple. Just like your other projects here General Services Application is your main project. Open mainAcitivity.java file present in the src folder and run it just like other apps. If you wish to deploy as an apk then right click on the folder and then select android tools and then deploy signed application.
